How to input problems in The Mathinator

On the main page there is a text field. This is where you can input the expression you want simplified or the equation you want solved.

Once you have inputted the problem press the "Solve / Simplify" button or press down the return key. After the problem is solved the orange page will scrunch up showing the solution beneath. To enter another problem either click "THE MATHINATOR" at the top of the page or press the return key.
