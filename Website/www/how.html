How to use The Mathinator

Let's say you wanted to solve for x in the equation: \[ {4*{({9*x}+{18*{\frac{x}{5}}})}}={{32*x}+6} \] Simply input : 4*(9*x+18*x/5)=32*x+6
click: Solve/Simplify

Or you wanted to simplify: \[ \frac{{{x}^{5}}*{{y}^{7}}*{{z}^{-2}}}{{{x}^{3}}*{{z}^{-10}}*y} \] Simplify input : (x^5*y^7*z^-2)/(x^3*z^-10*y)
click: Solve/Simplify

Output will show the answer and step-by-step the process in which solved.

The Mathinator is currently in alpha state. If you don't get what you want make sure of the following:

5x is 5*x
x^2/5+2 is (x^2)/5+2
